# exam08_提示詞練習重啟

這是一個 Python 專案範本，適合進行提示詞練習與重啟。

## 快速開始

1. 建立虛擬環境（uv 或 venv）或使用 Docker。
2. 安裝依賴：`pip install -r requirements.txt`（如有）。
3. 開始開發。

---

## 虛擬環境（uv）
```sh
python3 -m venv .venv
source .venv/bin/activate
```

或使用 [uv](https://github.com/astral-sh/uv)：
```sh
uv venv
source .venv/bin/activate
```

## Docker
如需 Docker 支援，可建立 `Dockerfile` 並撰寫對應內容。
